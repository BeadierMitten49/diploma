- # Списки
	- ### Таблица расчета затрат материалов (крафт)
		- Наименование продукта
		- Наименование сырья
		- Количество сырья на штуку продукта
	
	- ### Наименования упаковки
		- Наименование
		- Критический остаток
	
	- ### Наименования сырья
		- Наименование
		- Критический остаток
		- срок годности
	
	- ### Наименования продукции
		- Наименование
		- Критический остаток
		- срок годности
	
	- ### Фасовка
		- Наименование продукта
		- Количество штук продукта в коробке
	
	- ### Процент брака
		- Общий процент на все производство (в будущем можно сделать настройку для каждого отдельного сырья)
	
	- ### Заказчики
		- Наименование
		- Адрес
		- Комментарий

- # Склады
	- ### Упаковка
		 - Наименование (из списка)
		 - Количество
		 - Критический остаток (из списка)
		 - Комментарий
	
	- ### Сырье
		- Наименование (из списка)
		- Количество (в кг)
		- Дата поступления
		- Дата окончания срока годности (расчетное из списка)
		- Критический остаток (из списка)
		- Комментарий
	
	- ### Продукция
		- Наименование (из списка)
		- Количество (отображение в коробках по формуле + остаток в штуках, хранение в бд в штуках)
		- Номер партии 
		- Дата поступления
		- Дата окончания срока годности (расчетное из списка)
		- Критический остаток (из списка)
		- Комментарий
